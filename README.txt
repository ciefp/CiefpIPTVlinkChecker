Ciefp IPTV Link Checker
Ova web aplikacija omogućava proveru IPTV linkova iz JSON fajla. 
Aplikacija podržava MAG i M3U linkove, proverava njihovu dostupnost (status) i prikazuje rezultate u tabeli.

Glavne funkcionalnosti:
Provera linkova:

Aplikacija učitava JSON fajl sa IPTV linkovima.
Za svaki link proverava dostupnost pomoću HTTP GET zahteva.
Prikazuje status linkova (Available, Unavailable, Error).
Filteri:

Filtrirajte rezultate prema statusu (Available, Unavailable, Error) i tipu linka (MAG, M3U).
Eksport:

CSV Export: Omogućava izvoz odabranih (filtriranih) linkova u CSV fajl.
JSON Export: Omogućava izvoz samo dostupnih (Available) linkova u JSON fajl.
Progres bar:

Vizuelni prikaz napretka tokom provere linkova.
Prekid provere:

Omogućava korisniku da zaustavi proveru linkova u bilo kojem trenutku.
Upotreba:
Učitajte JSON fajl sa IPTV linkovima.
Kliknite "Check Links" za početak provere.
Filterišite rezultate prema želji.
Eksportujte rezultate u CSV ili JSON format pomoću odgovarajućih dugmadi.
Aplikacija je jednostavna i efikasna za proveru i izvoz IPTV linkova u željenom formatu.

Ciefp IPTV Link Checker
This web application allows checking IPTV links from a JSON file. 
The application supports MAG and M3U links, checks their availability (status) and displays the results in a table.

Main functionalities:
Checking links:

The application loads a JSON file with IPTV links.
For each link, it checks availability using an HTTP GET request.
It shows the status of the links (Available, Unavailable, Error).
Filters:

Filter results by status (Available, Unavailable, Error) and link type (MAG, M3U).
Export:

CSV Export: Enables the export of selected (filtered) links to a CSV file.
JSON Export: Enables the export of only available (Available) links to a JSON file.
Progress bar:

Visual display of progress during link checking.
Termination of verification:

Allows the user to stop checking links at any time.
Using:
Load the JSON file with IPTV links.
Click "Check Links" to start the check.
Filter the results as desired.
Export the results in CSV or JSON format using the corresponding buttons.
The application is simple and efficient for checking and exporting IPTV links in the desired format.
